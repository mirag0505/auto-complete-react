# React + TypeScript + Vite (auto-complete-component)

Link: gitpages
<https://mirag0505.github.io/auto-complete-react/>

## Dependency installation

```
pnpm install
```

## Dev start

```
pnpm dev
```

## Run test

```
pnpm test
```

## Run build

```
pnpm ci
```

The App consists:

Fetch ==> useQuery ==> Select --> Options -> Highlighted
                              --> Input
